# PracticalDataScience
This is the GitHub repository for the Practical Data Science course at Pace University.

There will be 2 projects completed during this course

## Project 1: URL Validator
Under the Project1 folder you will find the 2 deliverables 
Deliverable 1: includes various functions for calculating the validity of URL's

Deliverable 2: Includes application putting together query and URL validation

Readme.md in Project 1 folder provides the link to the application and more information

## Project 2: TinyTroupe
Under project 2 you will find 2 files explaining the project and how to replicate

Project uses OpenAI to have 2 AI bots communicate with one another.
